### 1. 查看课程大纲
首先阅读 [LangChain课程教学大纲与学习计划.md](./LangChain课程教学大纲与学习计划.md)，了解：

- 学习路径建议
- 技术栈选择
- 学习检查清单

### 2. 选择学习路径

**快速入门（2-3周）**
```
第01章 → 第02章 → 第03章 → 第05章 → 第06章 → 第12章
```

**标准学习（6-8周）**
```
按章节顺序学习第01-15章
```

**深度学习（10-12周）**
```
学习所有21个章节，包括理论和实践
```

### 3. 开始学习
1. 打开对应章节的 Markdown 文件
2. 按照章节内容学习视频
3. 完成实践项目
4. 记录学习笔记

## 💻 技术栈

### 核心框架
- Python 3.10+
- LangChain 1.0+

- Langfuse（追踪）

### 模型部署
- vLLM + Qwen3（推荐）
- Ollama（开发）

### 网页爬取
- Playwright（推荐）
- Selenium

### 向量数据库
- ChromaDB（本地）
- PG（本地）
