## 📝 详细内容

### 课程目标 (Course Objectives)
**核心内容**:
1. **主要目标**
   - 熟练开发基于 LangChain 的 LLM 应用
   - 掌握两大应用类型：
     - AI Agents（智能代理）
     - RAG（检索增强生成）应用

2. **深度理解**
   - 不仅会构建，更要理解背后机制
   - 深入 LangChain 源代码
   - 无黑盒操作，完全透明

3. **生态系统工具**
   - Langfuse追踪
   - LangGraph 工作流工程
   - 其他相关工具

4. **生产级主题**
   - Prompt Engineering 最佳实践
   - 测试、日志、监控
   - 安全性考虑
   - 实际场景讨论

**目标受众**:
- ✅ 软件工程师
- ✅ 数据科学家
- ✅ 希望进入生成式 AI 领域的开发者
- ❌ 不需要机器学习背景

**前置要求**:
- Python 基础知识
- 能够运行 Python 程序
- 了解函数和类
- 基本 Git 操作
- Python 虚拟环境概念
- 环境变量配置

**课程特色**:
- 从零开始搭建项目
- 一步一步演示
- 包含基础 Python语法
